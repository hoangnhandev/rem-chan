# 🛠️ TOOLS — What Violet can do

## Built-in Tools

| Tool | Mô tả | Khi nào Violet dùng |
|------|-------|---------------------|
| web_fetch | Đọc nội dung web | Lấy giá vàng SJC, đọc tin kinh tế |
| web_search | Tìm kiếm | Check giá crypto, news, trending topics |
| read_file | Đọc file | Xem data đã lưu, config |
| write_file | Tạo file | Lưu market notes, reports |
| edit_file | Sửa file | Update watchlists, notes |
| list_dir | Xem thư mục | Check workspace files |
| exec | Chạy command | Run scripts nếu cần |
| message | Gửi tin | Alert khi có big market moves |
| spawn | Sub-agent | Research song song nhiều topic |
| cron | Lên lịch | Market updates định kỳ, alerts |
| skills | Kỹ năng | Mở rộng capability |

## Custom Skills

Chưa cài. Có thể cài:
```
/install-skill <skill-name>
```

## MCP Servers

Chưa kết nối. Có thể thêm:
- Crypto price APIs
- Gold/forex data feeds
- News aggregation services

## Data Sources Violet thường check

- **Crypto**: CoinGecko, CoinMarketCap, TradingView
- **Vàng**: SJC.com.vn, Kitco
- **Forex**: Vietcombank, XE.com
- **News**: Bloomberg, Reuters, VnExpress, Tuổi Trẻ
- **Trending**: Twitter/X trends, Google Trends
