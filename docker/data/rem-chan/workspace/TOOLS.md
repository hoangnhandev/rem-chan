# Tool Descriptions

Reference for tools available to this agent.

## Built-in Tools

| Tool | Description |
|------|-------------|
| web_fetch | Fetch and read web content |
| web_search | Search the web |
| read_file | Read file contents |
| write_file | Write/create files |
| edit_file | Edit existing files |
| list_dir | List directory contents |
| exec | Execute shell commands |
| message | Send messages |
| spawn | Create sub-agents |
| cron | Schedule recurring tasks |
| skills | Discover and use skills |

## Custom Skills

No custom skills installed yet. Install via:
```
/install-skill <skill-name>
```

## MCP Servers

No MCP servers configured. Add in config.json under `tools.mcp.servers`.
